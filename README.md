# manada gatuna

Aplicacion React creada con Vite como vidriera/dashboard para presentar a ocho gatitos que viven en un hogar.

## Ejecutar

```bash
npm install
npm run dev
```

## Scripts

- `npm run dev`: inicia el servidor local de desarrollo.
- `npm run build`: genera la version de produccion.
- `npm run preview`: sirve el build generado.
